Star

An open standard for defining cloud native apps and building app-centric platforms.

Open Application Model [OAM] describes a runtime-agnostic approach to create platforms where developers focus on code and applications only.

Open Application Model is highly extensible to bring in any application operation features or abstractions.

With standard interfaces for infrastructure operators to declaring, customizing, and maintaining the underlying capabilities that are available on the platform.
